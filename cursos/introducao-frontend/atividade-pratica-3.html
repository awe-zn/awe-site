CSS + HTML

Para realizar esta tarefa você vai seguir os seguintes passos:

Para fazer, Siga os passos:

  1. Acesse este link e veja como deverá ficar sua atividade
  2. Em HTML foram utilizadas as seguintes tags (em sequência): 
     h1, p, div, a, h2, h3, h4, i;
  3. Use as seguintes cores para esta tarefa:
    #616db3
    #FFFFFF
    #AAAAAA

Para entregar, Siga os passos:

  1. Você deve atualizar o repositório local usando 
    git pull
  2. se não tiver na sua branch (com seu nome), deverá mudar para sua branch (com seu nome) usando: 
    git checkout sua-branch
  3. Você deverá criar um novo arquivo e salvá-lo com a seguinte nomenclatura:

    pratica-css1.5-basico.html

  4. Após realizar a montagem (em anexo), faça um 
    git add .
  5. O commit para esta atividade é: "CSS - Adicionando um novo arquivo com desafio de formatação CSS."; 
    git commit -m "CSS - Adicionando um novo arquivo com desafio de formatação CSS"
    git push
  6. Para entrega, você deverá realizar por meio do Github, copiando o link do seu commit correspondente;
  7. Após fazer o commit e push e verificar que o github atualizou, marque como entregue no Sala de Aula.
