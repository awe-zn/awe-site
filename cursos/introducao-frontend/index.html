Desafio final

O curso de Introdução Frontend tem por finalidade mostrar de forma básica, as maravilhas que podem ser feitas usando a dupla HTML + CSS. No fim do curso, caso aluno obtenha o nível mínimo de aproveitamento (70%), será declarado como um jovem Padawan.

O que será visto no curso

HTML

Sintaxe, Títulos, Parágrafo, Links, Imagens, Tabelas, Formulários.

CSS

Sintaxe, Vinculando folhas de estilo, textos, cores, fontes, plano de fundo, dimensões, espaçamento, bordas, posicionamento, subclasses

Desafio final

Desenvolvimento de uma página, que deverá ser implementada em sala de aula.
A página só será revelada no último dia.
