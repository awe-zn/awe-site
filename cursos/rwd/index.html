Curso de Web Design Responsivo
Desenvolvendo páginas para uma web universal.

Integrada à disciplina de Design Web e Arquitetura de Informação

Segundo a Teleco, o ano de 2015 fechou com mais de 7 Bilhões celulares vendidos no mundo. Dado que mostra como o mundo está se comportando e as tendências que devemos esperar para os próximos anos. A importância de planejar experiências para os diversos tipos de apresentações, em diversos tipos de aparelhos, deixou de ser uma perspectiva do universo estratégico para se tornar algo essencial.

Antenado com as tendências tecnológicas de vanguarda, desenvolvemos o curso de Web Design Responsivo (RWD).

Conteúdo programático

Aula 1 - Introdução

Introdução ao RWD;
Conceitos;
Estatísticas do mundo mobile;
Metodologias "m ponto" e "one web".

Atividade - Complementar 1

Desenvolvimento e um site, que será proposto em sala de aula. Necessário para prática de RWD.

Aula 2 - Príncipios de RWD e Layouts Fluídos

Surgimento do termo RWD;
Os 3 pilares da RWD;
Resoluções de telas;
Tipos de Medidas CSS;
Fórmula da RWD;
Layout fluxo e fixo;
Metatag viewport;
Converter layout fixo para layout fluído.

Atividade - Complementar 2

O site montado na atividade anterior será utilizado, trabalhando com os elementos fluídos.

Aula 3 - Recursos Flexíveis

CSS para imagens flexíveis;
CSS para outros recursos flexíveis;
Imagem e layout fluídos;
Técnicas para imagens em RWD;
Imagens em alta resolução;
Tipos de imagens para web.

Atividade - Complementar 3

Utilizando o site criado, iremos utilizar das técnicas de recursos flexíveis.

Aula 4 - Media Queries

Medias types;
Parâmetros para Medias Queries;
Operadores lógicos;
Breakpoints - pensando por fases;
Controle de erros;
Uso de media queries;
Media queries na prática.

Atividade - Complementar 4

Utilizaremos media queries para criar regras no nosso layout e melhorar sua experiência.

Aula 5 - Tópicos Especiais

Mobile First;
Pessoas e experiências mobile;
Importância do conteúdo;
Padrões de navegação;
Princípios de interface mobile.

Desafio final

Iremos colocar em prática os conhecimento trabalhados durante o curso. Será proposto um desafio para validar os conhecimentos.
