Atividade - Realizar montagem de site utilizando técnicas responsivas. Todas as imagens estão dentro da pasta img.

Da atividade

Você deverá realizar a montagem do site que se encontra em anexo no Google Sala de Aula. Importante lembrar das intruções abaixo:

  1. Baixo o arquivo em formato .zip;
  2. Descompacte na sua máquina;
  3. Renomeie a pasta para: Prova RWD - Seu nome;
  4. Realize a montagem do site (alterando os arquivos que estão dentro da pasta);
  5. Após finalizar, compacte a pasta e envie sua atividade;
  6. Torça para tudo está certo!

Regras

Abaixo seguem regras que deverão ser levadas em conta na montagem do seu site para que tudo ocorra bem.

  • Geral
    • Fontes: Lato (texto), Oswald (Títulos e Botões), FontAwesome (Ícones);
  • Layout Normal
    • Criar container para centralizar o bloco. Largura máxima de 900px;
  • Layout Responsivo
    • Criar regra para ajustar a tela a partir da largura de 500px;
    • Mudar imagem principal para img-principal-xs.jpg;
    • Criar bordas laterais, conforme a imagem ilustra.
