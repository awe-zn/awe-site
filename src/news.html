Publicado em 22.dez.2022

Agência Web Escolar lança edital para seleção de bolsistas do projeto

A Coordenação de Extensão lançou, nesta terça-feira (2), o Edital nº 4/2022-Coex/ZN, que trata da seleção para bolsista da Agência Web Escolar (AWE), projeto de extensão e prática profissional voltado ao desenvolvimento de habilidades técnicas e interpessoais dos estudantes do IFRN. A oferta é uma vaga para candidatos à bolsa, de R$ 300,00 mensais. O formulário de inscrição vai ficar disponível até 3 de agosto de 2022 no site da AWE: agenciaweb.ifrn.edu.br.

Publicado em 28.jul.2023

Extensão abre cadastro de interesse para Projeto Agência Web Escolar

A Coordenação de Extensão abriu, de hoje (28) até o dia 5 de agosto, no próprio setor, o cadastro de interesse para o Projeto Agência Web Escolar (AWE). No total, serão disponibilizadas 20 vagas, voltadas para microempreendedores individuais que possuam CNPJ, preferencialmente do bairro Potengi, visto que o principal objetivo do projeto é o desenvolvimento de websites para esse público-alvo do bairro.
